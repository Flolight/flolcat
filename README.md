<!-- ABOUT THE PROJECT -->
## About The Project

flolcat is a go implementation of the [lolcat](https://github.com/busyloop/lolcat) classic command
### Installation

```sh
go get github.com/flolight/lolcat
```



<!-- USAGE EXAMPLES -->
## Usage

echo "MyText" | flolcat
